#### 1、最简单的方法就是创建临时变量

```
let = 11,b = 22 ,t;
t = a;
a = b;
b = t;
```
#### 2、先建立两个变量的关系，变量求和

```
let a = 11,b = 22 ;
a = a + b;
b = a - b;
a = a - b;
```
#### 3、变量求差（原理同上）

```
let a = 11,b = 22 ;
a = a - b;
b = a + b;
a = b - a;
```
#### 4、先让其中一个变量变成对象，存储两个变量的值

```
let a = 11,b = 22 ;
a = {a:a,b:b};
b = a.a;
a = a.b;
```
#### 5、同上，先让其中一个变量变成数组，存储两个变量的值

```
let a = 11,b = 22 ;
a = [a,b];
b = a[0];
a = a[1];
```
#### 6、利用运算符优先级

```
let a = 11,b = 22 ;
a = [b,b=a][0];
```
#### 7、利用ES6解构赋值

```
let a = 11,b = 22 ;
[a,b] = [b,a];
```
#### 8、利用异或运算符^(原理与2、3一样)

```
let a = 11,b = 22 ;
a = a^b;
b = a^b;
a = a^b;
```
